1)Batch operating system

This type of operating system does not interact with the computer directly. There is a worker who occupies similar jobs that are similar in requirements and group them in groups. The operator is responsible for sorting jobs with similar needs. Examples of batch-based operating system: payroll system and bank statement

2. Timeshare operating systems

Each task takes a fraction of time to execute, as all tasks run smoothly. Also, each user gets CPU time when using one single system. Also, these systems are called (multitasking) systems. The job is allowed to be from one user or from different users.

3)Distributed operating system

This type of operating system is the latest development in all parts of the computer technology world and is widely used in the world, very quickly. Various independent interconnected computers communicate with each other by using a shared network
